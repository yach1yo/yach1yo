# Yach1yo 个人主页

我的研究项目与 Vibe Coding 试作品展示。

## 功能

- **左侧栏 (1/4)**：个人头像、简介、技能标签、社交链接
- **右侧栏 (3/4)**：
  - 📚 研究项目 — 学术研究、论文进展、实验成果
  - ⚡ Vibe Coding 试作品 — 创意编程、Demo 和快速原型
- 支持文件上传（附件存储于 localStorage）
- 支持在线编辑文字内容
- 数据自动保存至浏览器本地
- 一键导出 JSON 备份

## 使用方式

直接用浏览器打开 `index.html`，或部署至 GitHub Pages。
